# Функціонал з управління статтями

Реалізовано:

- 1 Створення статей.
- 2 Редагування кожної статті (оновлення).
- 3 Видалення статей.

## Стек технологій:

React + TypeScript + Node + Express + MongoDB

### Запустити проект

- npm run dev (frontend)
- npm run server (backend)
